UX/UI Web & Mobile Applications


Project Name: Dino's Tomato Pie Redesign

Participants: 1 | Role: UX/UI Designer | Programs: InVision, AdobeXD, Adobe Photoshop | Description:
Redesigned a website from a well known pizza restaurant in Seattle WA, called Dinos Tomato Pie.
The original design does not represent the friendly and inviting atmophere.
The original website is chaotic, unappealing, and may be irritable to some users that are sensitive to flash animation.
Using restaurant's original content, application is designed to be easily accessible on various platforms with an appealing interactive design.
To demonstrate a friendly and social atmosphere, the story page was created which is a social page between the restaurant's staff and their customers.

Project Name: GoogleMaps Mobile Redesign

Participants: 1 | Role: UX/UI Designer | Programs: InVision & Adobe Illustrator | Description:
A self-challenge to improve GoogleMaps Application with desirable user-friendly features and suitable visual design.
Redesigned navigation path with clearer indicators such as arrows pointing to the destination instead of dots as the original's design.
Added parking garage features when user uses saved parking option with accesibility and effeciency in mind.
All new features designed with a suitable visual design to fit in with the Original GoogleMaps Application.


Project Name: Virtual Point Defiance Zoo and Aquarium

Participants: 3 | Role: Web Designer & Developer | Programming Languages: HTML/CSS/JavaScript | Description:
A website application to virtually tour The Point Defiance Zoo & Aquarium.
Application was created within 3 days, and uses content from database and APIs for each animal (some APIs have a limit on requests and will show placeholder image).
Using map feature with coordinats in HTML and JavaScript I made a still image of a real Point Definace Zoo map into an interactive map.
The map has access to each animal on the map via a "click" or "tab" key from which users can further explore and learn about each animal.
The application also has a donation page where users receive framed badges in bronze, silver, or gold of the animals they donate to as a reward.
